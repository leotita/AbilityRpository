## 表格

### 基本表格
Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

语法格式如下：

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

eg:
| 年龄 | 性别 |
| - | - |
| 10 | 男 |
| 20 | 女 |

### 对齐方式

我们可以设置表格的对齐方式：

-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。
| 年龄 | 性别 | 名字 |
| -: | :- | :-: |
| 10 | 男 | 小明 |
| 20 | 女 | 小红 |
